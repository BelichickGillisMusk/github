# Migration Checklist: Squarespace → Cloudflare

## Content Inventory
- [ ] Homepage content and copy
- [ ] Service descriptions
- [ ] Contact information
- [ ] Images and media assets
- [ ] SEO metadata (titles, descriptions, keywords)
- [ ] Any forms or integrations
- [ ] Google Analytics / tracking codes
- [ ] Custom domain email settings

## DNS & Domain
- [ ] Current DNS records documented
- [ ] Domain registrar identified (Squarespace or external?)
- [ ] Cloudflare DNS zone created
- [ ] Nameservers updated (if needed)
- [ ] SSL/TLS configured on Cloudflare

## New Site Build
- [ ] Cloudflare Pages project created
- [ ] Site structure matches current pages
- [ ] All content migrated
- [ ] Mobile responsive verified
- [ ] Forms reconnected (contact, quote requests)
- [ ] Tracking/analytics reinstalled

## Go-Live
- [ ] Staging site reviewed and approved by Bryan
- [ ] DNS cutover executed
- [ ] Old Squarespace site kept as backup (30 days)
- [ ] All links and redirects working
- [ ] Squarespace subscription cancelled
