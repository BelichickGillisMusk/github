OBD Testing

$75 per vehicle

On-board diagnostic (OBD) testing for 2013+ heavy-duty diesel trucks. Required by CARB's Clean Truck Check program for vehicles over 14,000 lbs GVWR.

What's Included

  • Complete OBD scan using CARB-approved equipment
  • Emissions system diagnostic (NOx, PM, exhaust aftertreatment)
  • Malfunction Indicator Lamp (MIL) and fault code check
  • Official pass/fail documentation on-site
  • Free retest within 150 days if you fail

Who Needs This

  • Heavy-duty diesel trucks (2013 and newer, 14,000+ lbs GVWR)
  • Owner-operators and fleets meeting CARB Clean Truck Check requirements
  • Vehicles due for biennial testing or DMV registration compliance

Testing Process

  1. We arrive at your location with CARB-certified OBD equipment
  2. Connect to your vehicle's diagnostic port (10-15 minutes)
  3. Run complete emissions system diagnostic
  4. Provide official pass/fail documentation immediately
  5. If you fail, we explain the codes and offer free retest after repairs

Average test time: 15-20 minutes per vehicle

OVI-J1667 (Smoke / Opacity Testing)

$250 per vehicle

Opacity Visible Inspection per SAE J1667 standard — professional smoke/opacity testing for pre-2013 heavy-duty diesel trucks and vehicles exempt from OBD requirements.

What's Included

  • SAE J1667 compliant opacity measurement
  • Snap-acceleration smoke test procedure
  • Visual exhaust system inspection
  • Official CARB documentation on-site
  • Free retest within 150 days if you fail

Who Needs This

  • Pre-2013 heavy-duty diesel trucks (under Periodic Smoke Inspection Program)
  • Vehicles exempt from OBD testing (agricultural, construction, certain vintage)
  • Fleets with mixed-age equipment requiring PSIP compliance

Testing Process

  1. We arrive with certified opacity measurement equipment
  2. Warm up your vehicle to operating temperature
  3. Perform snap-acceleration test per J1667 protocol
  4. Measure exhaust opacity against CARB pass/fail thresholds
  5. Provide official documentation immediately

Average test time: 25-30 minutes per vehicle
